Exercise 2 MLE Theory result

Given the probability density of Poisson Distribution, the likelihood function is defined by \[ \delta_1=\frac{1}{n}\sum_{i=1}^NX_i \] by i.i.d Poisson, \[ \mathbb{E}[\delta_1]=\lambda\\ Var[\delta_1|\lambda]=\frac{1}{n^2}*nVar[X]=\frac{\lambda}{n} \] so the bias is \(0\) and the variance is \(\lambda\). Therefore, the MSE is \[ MSE=\lambda/n \]

Exercise 3 Bayes Theory MSE

Recall that the Bayes estimator under the squared error loss is the posterior mean. Using Gamma-Poisson Conjugate, the posterior distribution follows distribution \(Gamma(\alpha+\sum_i X_i, \beta+n)\), with mean \[ \delta_2=\frac{\alpha+\sum_i X_i}{\beta+n} \] Again, by iid Poisson, we know that \[ \mathbb{E}[\delta_2]-\lambda=\frac{\alpha+n\lambda}{\beta+n}-\lambda\\ Var[\delta_2|\lambda]=\frac{1}{(\beta+n)^2}*nVar[X]=\frac{n\lambda}{(\beta+n)^2} \] therefore, the MSE is \[ MSE=(\frac{\alpha+n\lambda}{\beta+n}-\lambda)^2+\frac{n\lambda}{(\beta+n)^2} \] Using the result in exercise 1, the L1 loss estimator is the median of posterior distribution. \[ \delta_3=median((Gamma(\alpha+\sum_i X_i, \beta+n)) \]

Exercise 4 Mean Square Error

# Set parameters for the simulation
a <- 6
b <- 2
sample_size <- 10  # Size of each sample
num_simulations <- 1000  # Number of simulations

mse1 <- c()
mse2<- c()
mse3<- c()


# Initialize a vector to store the squared errors
squared_errors1 <- numeric(num_simulations)
squared_errors2 <- numeric(num_simulations)
squared_errors3 <- numeric(num_simulations)

# Run the simulations
for (lambda in 1:5) { # True parameter value of the Poisson distribution
  for (i in 1:num_simulations) {
    # Generate a random sample from Poisson distribution
    sample <- rpois(sample_size, lambda)
    
    # Calculate the sample mean
    delta_1 <- mean(sample)
    delta_2 <- (a+sum(sample))/(b+10)
    delta_3 <- qgamma(0.5, shape = a+sum(sample), rate = b+10)
    
    # Compute the squared error
    squared_errors1[i] <- (delta_1 - lambda)^2
    squared_errors2[i] <- (delta_2 - lambda)^2
    squared_errors3[i] <- (delta_3 - lambda)^2
  }

  # Calculate the MSE
  
  mse1 <- c(mse1,mean(squared_errors1))
  mse2 <- c(mse2,mean(squared_errors2))
  mse3 <- c(mse3,mean(squared_errors3))
}
# Print the MAE
plot(c(1,2,3,4,5), mse1, type = 'l', col = 'blue', xlab = 'Lambda', ylab = 'MSE', main = 'MSE of Delta_1, Delta_2, and Delta_3')
lines(c(1,2,3,4,5), mse2, col = 'red')
lines(c(1,2,3,4,5), mse3, col = 'green')
legend('topright', legend = c('Delta_1', 'Delta_2', 'Delta_3'), col = c('blue', 'red', 'green'), lty = 1)

From the graph, clearly the \(delta_1\) behaves worse than the others in the case of MSE. Recall that \(delta_1\) is the only unbiased estimator, there shall be a tradeoff between bias and varience.

MAE is less sensitive to outliers than MSE. Each error contributes proportionately to the total amount of error, making it a good choice when you want to avoid giving extra weight to outlier errors.

Exercise 6 Huber Loss

# Set parameters for the simulation
a <- 6
b <- 2
sample_size <- 10  # Size of each sample
num_simulations <- 1000  # Number of simulations
ep <- 10


hub1 <- c()
hub2 <- c()
hub3 <- c()


# Initialize a vector to store the squared errors
hub_errors1 <- numeric(num_simulations)
hub_errors2 <- numeric(num_simulations)
hub_errors3 <- numeric(num_simulations)

# Run the simulations
for (lambda in 1:5) { # True parameter value of the Poisson distribution
  for (i in 1:num_simulations) {
    # Generate a random sample from Poisson distribution
    sample <- rpois(sample_size, lambda)
    
    # Calculate the sample mean
    delta_1 <- mean(sample)
    delta_2 <- (a+sum(sample))/(b+10)
    delta_3 <- qgamma(0.5, shape = a+sum(sample), rate = b+10)
    
    # Compute the squared error
    if(abs(delta_1 - lambda) <= ep){
    hub_errors1[i] <- 0.5*(delta_1 - lambda)^2}
      else{
    hub_errors1[i] <- ep*abs(delta_1 - lambda)-0.5*ep^2}
    
      # 2
     if(abs(delta_2 - lambda) <= ep){
    hub_errors2[i] <- 0.5*(delta_2 - lambda)^2}
      else{
    hub_errors2[i] <- ep*abs(delta_2 - lambda)-0.5*ep^2}
    
      # 3
     if(abs(delta_3 - lambda) <= ep){
    hub_errors3[i] <- 0.5*(delta_3 - lambda)^2}
      else{
    hub_errors3[i] <- ep*abs(delta_3 - lambda)-0.5*ep^2}
  }


  

  # Calculate the hub
  
  hub1 <- c(hub1,mean(hub_errors1))
  hub2 <- c(hub2,mean(hub_errors2))
  hub3 <- c(hub3,mean(hub_errors3))
}
# Print the HUB
plot(c(1,2,3,4,5), hub1, type = 'l', col = 'blue', xlab = 'Lambda', ylab = 'MSE', main = 'HUB of Delta_1, Delta_2, and Delta_3')
lines(c(1,2,3,4,5), hub2, col = 'red')
lines(c(1,2,3,4,5), hub3, col = 'green')
legend('topright', legend = c('Delta_1', 'Delta_2', 'Delta_3'), col = c('blue', 'red', 'green'), lty = 1)

taking the average over MSE and MAE. when epsilon goes larger, you penalize more.
